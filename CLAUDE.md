# PLIALU — Calculette

Projet web statique de façonnage aluminium sur mesure.

## Fichiers principaux

- `calcul.html` — Calculette de prix (DVP, matière, RAL, épaisseur)
- `devis.html` — Génération de devis PDF (jsPDF + pdf-lib)
- `dessinateur.html` — Zone de dessin canvas avec textes redimensionnables
- `index.html` — Page d'accueil

## Règles

- Pas de framework, pas de build : HTML/CSS/JS vanilla uniquement
- Données persistées via `localStorage` (clés : `plialu-devis`, `plialu-devis-dessins`)
- GitHub Pages déploie depuis `main`
- Développer sur une branche `claude/...` puis merger via PR
