Aide — Outil de dessin

Comment utiliser le configurateur ?

  • Cliquez sur un segment pour modifier sa longueur (en mm).
  • Cliquez sur un angle pour modifier sa valeur (min 45°).
  • Utilisez les boutons Ajouter un segment pour créer de nouveaux plis.
  • Le bouton Centrer recentre le dessin dans la zone de travail.

Le développé maximum est de 1 500 mm. La longueur minimale par segment est de 15 mm.

Comment saisir une dimension ?

Cliquez directement sur le chiffre affiché sur le segment dans le dessin, ou utilisez le champ de saisie correspondant dans le panneau latéral.

Toutes les dimensions sont exprimées en millimètres (mm).

Comment calculer la longueur ?

La longueur développée est la somme de tous les segments du profil à plat, sans les épaisseurs de pliage.

La longueur de coupe (longueur du profilé) est la cote que vous indiquez dans le champ Longueur — c'est la dimension du profilé fini dans sa direction longitudinale.

Qu'est-ce que la face laquée ?

Pour les finitions prélaquées, la peinture est appliquée sur une seule face de la tôle (face visible).

Indiquez quelle face sera visible une fois le profilé posé. Le configurateur en tient compte pour orienter correctement le pli.

Comment choisir l'épaisseur ?
ÉpaisseurUsage recommandé
8/10 (0,8 mm)Bardage léger, protections intérieures
10/10 (1,0 mm)Couvertines, bavettes standards
15/10 (1,5 mm)Acrotères, profils structurels — recommandé

En cas de doute, le 15/10 offre un bon équilibre rigidité / poids pour la plupart des applications.
